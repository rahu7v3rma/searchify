# Softools - SEO tools

## Tech Stack

- Language: TypeScript
- Backend: Next
- Database: Supabase
- Frontend: React, Tailwind
- Testing: Jest, React-Testing-library
- Linting: ESLint
- Formatting: Prettier

## Run the app:

```bash
npm run dev
```

## Run the tests:

```bash
npm run test
```

## Lint the code:

```bash
npm run lint
```

## Build the project:

```bash
npm run build
```
